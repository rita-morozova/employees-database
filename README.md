# Employees Database

## Description
An employees database built using a microservices architechture. A user can add/edit/delete employees from the database.

## Technologies used:
 - Spring Boot
 - Mongo DB
 - Docker
 - Kubernetes
 - Apache Kafka
 - API Gateway
 - Service Discovery
 - React
 - Angular
 - JSON Web Tokens
 - Google Cloud
